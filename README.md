# Ticket Dashboard Project

A modern Ticket Dashboard application built with Next.js, React, TypeScript, and MongoDB.

## 🚀 Features

- Create new tickets
- Edit existing tickets
- Delete tickets
- List and manage all tickets
- Dashboard interface with charts
- Responsive and clean UI

## 🛠️ Technologies Used

- Next.js
- React
- TypeScript
- MongoDB
- Mongoose
- Tailwind CSS
- Chart.js
- React Chartjs 2
- Lucide React
